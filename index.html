Open Data Portal and Web Apps

This page shows examples of how Detroit's Open Data Portal as well as web-based applications could be better integrated in the menus of Detroitmi.gov in order to be more accessible to users. There is also a plan to create a new internal page in Detroitmi.gov that would showcase and link to all available web apps.

Additions to the Hamburger Menu

In the main menu of Detroitmi.gov, we'd add 2 additonal links. One external link that would lead to the ESRI Open Data Portal, and one that would link to a newly created internal Detroitmi.gov page which houses all web apps.

Additions to the "Working for You" Menu

In addition to the 2 links added to the hamburger menu, we would also add a Technology section to the Working for You interactive menu on the front page. From this section, we would place the same links to the external Open Data Portal and the internal Web Apps page.

New Web Apps Page

The newly created Web Apps page would house links and brief details about all the web apps available citywide, including Open Data apps, Web Team apps, and third party apps. This page would give users one landing spot to be able to see all the web apps available to our citizens, as well as provide brief detail about what the app does..
